Questions

  1. What is a dictionary

    2. Simple enough. I use dictionaries every day. It is a list of key value pairs. Keys are a string, tuple, or a number that point to a value. I got this wrong because I said they couldn't be numbers when he asked me. I have never worked with keys as numbers. Keys are unique and immutable.

    You can add, modify, or delete key-value pairs in a dictionary, making it a versatile data structure for various tasks like data storage, retrieval, and manipulation in Python.

  2. What is the ORM used for?

    3. Object-Relational Mapping (ORM) is used to map objects in your code to tables in a relational database. It provides an abstraction layer that allows you to interact with the database using objects and classes instead of writing raw SQL queries. This simplifies database operations and makes it easier to work with databases in your software applications.

  3. What is a lambda?

    4. In Python, a lambda function is an anonymous, small, and inline function that can have any number of arguments but can only have one expression. Lambda functions are typically used for short, simple operations where you don't want to define a full function using the `def` keyword. They are often used as arguments to higher-order functions like `map()`, `filter()`, and `sorted()`.

  4. What are decorators used for?

    5. Decorators in Python are used to modify or enhance the behavior of functions or methods. They are typically applied to functions using the "@" symbol and are commonly used for tasks like logging, authorization, caching, and more. Decorators allow you to add functionality to functions without modifying their code directly, promoting code reusability and separation of concerns.

  5. Why would you use a list versus a tuple?

    6. Lists and tuples in Python are both used to store collections of items, but there are key differences between them. Lists are mutable, meaning you can add, remove, or modify elements after creation. Tuples, on the other hand, are immutable, and their elements cannot be changed once they are defined. You would use a list when you need a collection of items that may change during the program's execution. Tuples are useful when you want to create a collection of items that should not change, such as coordinates or configuration settings.

  6. Why are tuples immutable?

    7. Tuples are immutable in Python to ensure their integrity and make them suitable for use as keys in dictionaries or elements in sets. Immutability guarantees that once a tuple is created, its elements cannot be accidentally or maliciously modified, making it safe for certain use cases where data should remain constant. This immutability also allows tuples to be hashable, which is necessary for dictionary keys and set elements.

  7. How can you efficiently use Python to count the number of times a value appears in a list?

    8. You can efficiently count the number of times a value appears in a list using the `count()` method provided by Python lists. Here's an example: 

            
my_list = [1, 2, 2, 3, 4, 2, 5, 2]
value_to_count = 2

count = my_list.count(value_to_count)
print(f"The value {value_to_count} appears {count} times in the list.")
    This code snippet will count the occurrences of value_to_count in my_list and print the result.
